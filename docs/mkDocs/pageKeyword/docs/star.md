# 星号 `*` 用法（Markdown 中的多功能修饰符）

**星号 `*` 是 Markdown 最核心的文本修饰标记之一，覆盖“斜体、加粗、无序列表、特殊嵌套”等多种常用语法，是读写 MarkDown 必掌握的基础。**

---

!!! note "读法"
    - 英文：asterisk /ˈæstərɪsk/
    - 日语：アスタリスク
    - 常见误区："*" 前后不能有空格、与数学中的乘号不同

---

!!! info "用途"
    - 文本**斜体、粗体（加粗）、加粗斜体**装饰
    - 创建**无序列表**
    - 与其它标记组合，嵌套高阶格式
    - 显式文本中的星号（转义）
    - 支持各种场景下的 Markdown 美化写作

---

<h2>基本语法一览</h2>

| 用法       | 标记           | 结果                     |
|:-----------|:---------------|:------------------------|
| 斜体       | `*斜体*`       | *斜体*                   |
| 加粗       | `**加粗**`     | **加粗**                 |
| 又粗又斜   | `***粗斜***`   | ***粗斜***               |
| 无序列表   | `* 项目1`      | <ul><li>项目1</li></ul>  |
| 转义星号   | `\*普通星号\*` | \*普通星号\*             |

---

=== "斜体"
    !!! example "示例"
        - 写法：`*这是斜体*`
        - 效果：*这是斜体*

    !!! tip "注意"
        - 左右星号需紧贴文本，**不能有空格**
        - 可和下划线 `_` 等价互换

=== "加粗"
    !!! example "示例"
        - 写法：`**加粗**`
        - 效果：**加粗**
    !!! tip "常见错误"
        - `** 加粗 **` ← 不能有空格，写成这样无效果

=== "斜体+加粗"
    !!! example "示例"
        - 写法：`***又粗又斜***`
        - 效果：***又粗又斜***

    !!! info "用法"
        - 支持多层组合，例如 `****多重加粗斜****` 也能显示。
        - 推荐最多用三层，否则难以区分。

=== "无序列表"
    !!! example "标准写法"
        ```
        * 项目A
        * 项目B
        ```
        - 渲染效果如下： 
 
          * 项目A
          * 项目B

    !!! tip "列表嵌套也常见"
        ```
        * 一级
            * 二级
                * 三级
        ```

=== "转义星号（防止变格式）"
    !!! example "写法"
        - 要输出真正的星号，用反斜杠前缀：`\*`
        - 示例：  
            ```
            \*普通星号，不是斜体*
            ```
            渲染：\*普通星号，不是斜体*

---

<h2>进阶用法</h2>

=== "嵌套修饰"
    !!! example "复杂样例"
        ```
        **加粗 *里面嵌套斜体***
        ```
        效果：**加粗 *里面嵌套斜体***
    !!! info "顺序灵活"
        - 可以先斜体再加粗，也可以反过来，Markdown 渲染器一般都能兼容

=== "搭配其它符号"
    !!! example "斜体+代码块"
        ```
        *`斜体里的代码`*
        ```
        效果：*`斜体里的代码`*

    !!! example "列表与斜体混用"
        ```
        * *斜体列表*
        * **加粗列表**
        ```
        渲染：

        * *斜体列表*
        * **加粗列表**

=== "文本星号显示"
    !!! example "数学公式与星号"
        - 公式如 `f(x) = x * x`，可写为
        ```
        f(x) = x \* x
        ```
        输出：f(x) = x \* x

---

!!! info "复杂嵌套"
    - Markdown 渲染器（如 MkDocs Material、Typora等）几乎都支持星号各种合法组合嵌套
    - 但有些极特殊格式和 hack 写法，在 GitHub 等原生渲染器不通用

---

!!! warning "注意点"
    - 加粗、斜体等多级嵌套时，编辑器/渲染器表现可能略有不同  
    - 不同平台如 GitHub、Gitee、Typora、VSCode、MkDocs Material，解析细节略异，建议多测试主流场景
