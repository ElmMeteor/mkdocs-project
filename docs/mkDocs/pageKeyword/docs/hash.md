# 标题 `#`

**# 是 Markdown 的 标题标记（heading marker）**

!!! note "读法"
    英语：“hash” <br>
    日语：ハッシュ

!!! info "标题语法(原生语法)"
    **标记标题，生成 HTML 对应的 `<h1> ~ <h6>`**  
    **用于 自动生成导航栏（TOC / Table of Contents）**  
    **在 MkDocs里，# 对应页面内锚点，方便跳转**

    * 每页 H1 只能有一个,多个 H1 会影响页面标题显示和 TOC
    * 导航自动生成依赖 H2/H3，H4/H5/H6 默认不加入导航栏，但是会生成页内跳转

=== "code"
    ```yaml
    # 标题 `#`
    ## 标题 `##`
    ### 标题 `###`
    #### 标题 `####`
    ##### 标题 `#####`
    ###### 标题 `######`
    ```
