# 概要

MkDocs 是一个用来“把 Markdown 文档自动生成网站”的工具。

简易流程：
➡ 制作 Markdown 文件（.md） → MkDocs 自动生成一个可搜索的文档网页。


=== "Markdown文件(.md)"
    ```
    # 介绍

    MkDocs 是一个用来“把 Markdown 文档自动生成网站”的工具。

    流程：
    ➡ 制作 Markdown 文件（.md） → MkDocs 自动生成一个可搜索的文档网页。
    ```
